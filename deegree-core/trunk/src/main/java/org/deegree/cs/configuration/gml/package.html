This package contains the classes used for reading gml based coordinate system configuration backends. The backends are modelled by supplying a Resource. This resource is responsible for resolving xlinks in the gml representation of a Coordinate system (component).

The default crs configuration can be changed by editing the org.deegree.cs.configuration.crs_providers.properties file, the default provider is {@link org.deegree.cs.configuration.deegree.xml.DeegreeCRSProvider}.
